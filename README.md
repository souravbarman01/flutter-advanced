# flutter_advanced

A **Flutter project for experimenting with advanced features and best practices in mobile application development**.

---

## Flutter Version
`Flutter 3.38.5`

---

# Project Goal

The primary goal of this project is to implement a **product flow system**, including:

- Product listing
- Add to cart functionality
- Checkout process

This project focuses on:

- Scalable architecture
- Clean and maintainable code
- Efficient routing
- Responsive UI
- Proper linting practices

---

# Dart Concepts Practiced

This project also focuses on improving core **Dart programming skills**, including:

- Generics
- Mixins
- Factory constructors
- Object-Oriented Programming (OOP)
- Permission handling
